1. ОБЩИЕ ПОЛОЖЕНИЯ

Настоящая Политика конфиденциальности (далее — Политика) определяет порядок сбора, обработки, хранения, использования и защиты персональных данных работников при использовании мобильного приложения AvaTracker.

Политика разработана в соответствии с:
Законом Республики Казахстан «О персональных данных и их защите»;
Трудовым кодексом Республики Казахстан;
Законом Республики Казахстан «Об информатизации»;
иными нормативными правовыми актами Республики Казахстан, регулирующими вопросы обработки и защиты персональных данных.

Использование мобильного приложения AvaTracker означает ознакомление работника с настоящей Политикой и согласие с условиями обработки его персональных данных в случаях, когда получение такого согласия требуется законодательством Республики Казахстан.

Настоящая Политика распространяется исключительно на мобильное приложение AvaTracker и не регулирует обработку персональных данных сторонними сервисами или интернет-ресурсами, ссылки на которые могут содержаться в приложении.

2. О ПРИЛОЖЕНИИ

AvaTracker является корпоративной информационной системой работодателя, предназначенной для автоматизации процессов учета рабочего времени работников.

Приложение используется исключительно в целях:
регистрации времени прихода на работу;
регистрации времени ухода с работы;
фиксации фактического присутствия работника;
учета рабочего времени;
формирования табелей учета рабочего времени;
контроля соблюдения трудовой дисциплины;
анализа опозданий и иных показателей трудовой дисциплины;
идентификации личности работника при фиксации отметок.

Приложение не предназначено для постоянного отслеживания местоположения работников, контроля их личной жизни либо использования персональных данных в рекламных, маркетинговых или иных целях, не связанных с исполнением трудовых отношений.

3. ПРАВОВЫЕ ОСНОВАНИЯ ОБРАБОТКИ ПЕРСОНАЛЬНЫХ ДАННЫХ

Обработка персональных данных осуществляется Оператором на следующих правовых основаниях:
исполнение обязанностей работодателя, предусмотренных трудовым законодательством Республики Казахстан;
исполнение трудового договора, заключенного между работодателем и работником;
соблюдение требований законодательства Республики Казахстан;
согласие субъекта персональных данных — в случаях, когда получение такого согласия предусмотрено законодательством Республики Казахстан;
обеспечение функционирования корпоративной информационной системы учета рабочего времени.

Обработка персональных данных осуществляется только в объеме, необходимом для достижения целей, указанных в настоящей Политике.

4. КАКИЕ ПЕРСОНАЛЬНЫЕ ДАННЫЕ ОБРАБАТЫВАЮТСЯ

В зависимости от функционала приложения Оператор может осуществлять обработку следующих персональных данных работника:

Идентификационные данные
фамилия;
имя;
отчество (при наличии);
индивидуальный идентификационный номер (ИИН);
номер мобильного телефона;
идентификатор учетной записи.

Кадровые сведения
должность;
структурное подразделение;
филиал;
статус работника;
сведения, необходимые для функционирования системы учета рабочего времени.

Данные учета рабочего времени
дата отметки;
время отметки;
тип отметки (приход/уход);
история отметок;
сведения о нарушениях режима рабочего времени;
аналитическая информация, формируемая системой на основании произведенных отметок.

Геолокационные данные
При выполнении отметки прихода либо ухода приложение может получать сведения о текущем местоположении мобильного устройства работника.
Геолокация используется исключительно в момент фиксации отметки для подтверждения нахождения работника в установленной работодателем рабочей зоне.
Приложение не осуществляет постоянное либо фоновое отслеживание местоположения работника.

Биометрические персональные данные
Для подтверждения личности работника при выполнении отметки приложение может получать фотографию лица работника.
Фотография лица относится к биометрическим персональным данным и используется исключительно:
для подтверждения личности;
предотвращения передачи учетной записи третьим лицам;
предотвращения фиктивных отметок;
обеспечения достоверности учета рабочего времени.

Биометрические данные:
не используются для маркетинга;
не используются для рекламы;
не используются для построения поведенческих профилей;
не используются для автоматического распознавания личности вне целей, предусмотренных настоящей Политикой.

Изображение с камеры
При использовании функции сканирования QR-кода приложение получает изображение с камеры мобильного устройства исключительно в объеме, необходимом для распознавания QR-кода.
Изображение не используется в иных целях.

Технические данные
В целях обеспечения стабильной работы приложения могут автоматически собираться:
модель устройства;
версия операционной системы;
версия приложения;
журналы ошибок (Crash Logs);
диагностическая информация;
сведения о работе приложения;
информация о сетевых ошибках;
технические идентификаторы, необходимые для функционирования приложения.

Указанные сведения используются исключительно для обеспечения работоспособности приложения, устранения технических ошибок и повышения качества сервиса.

Оператор осуществляет обработку персональных данных с соблюдением принципов законности, справедливости, соразмерности и минимизации объема обрабатываемых данных.

Индивидуальный идентификационный номер (ИИН) используется для однозначной идентификации работника в информационной системе работодателя.

Номер мобильного телефона используется для авторизации, восстановления доступа к учетной записи и взаимодействия с пользователем приложения.

Фамилия, имя, отчество, должность, подразделение и филиал используются исключительно для идентификации работника и отображения информации в корпоративной системе учета рабочего времени.

Данные учета рабочего времени используются для формирования табелей учета рабочего времени, внутренней аналитики, контроля соблюдения трудовой дисциплины и исполнения работодателем обязанностей, предусмотренных трудовым законодательством Республики Казахстан.

Геолокация определяется исключительно в момент фиксации отметки прихода или ухода. Полученные сведения используются только для подтверждения нахождения работника в установленной работодателем рабочей зоне.

Приложение не осуществляет постоянный, периодический или фоновый сбор сведений о местоположении работника.

Фотография лица используется исключительно для подтверждения личности работника при фиксации отметки прихода либо ухода и предотвращения использования учетной записи третьими лицами.

Технические сведения используются исключительно для диагностики неисправностей, обеспечения информационной безопасности, улучшения производительности приложения и повышения качества обслуживания пользователей.

Оператор не принимает решений, имеющих юридические последствия для работника, исключительно на основании автоматизированной обработки персональных данных.
