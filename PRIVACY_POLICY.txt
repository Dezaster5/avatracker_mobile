О ПРИЛОЖЕНИИ

AvaTracker — корпоративное приложение для сотрудников компании.

Приложение используется для учёта рабочего времени, фиксации отметок
прихода и ухода, а также мониторинга опозданий.

Для работы приложение может обрабатывать:
—  ИИН сотрудника;
—  номер телефона;
—  ФИО, должность, отдел и филиал;
—  историю отметок прихода и ухода;
—  дату и время сканирования;
—  геолокацию во время отметки;
—  данные камеры при сканировании QR-кода;
—  биометрические данные (фотографию лица) для фото-верификации личности
   при отметке.

Данные используются только для внутреннего учёта рабочего времени и
проверки активности сотрудника.

------------------------------------------------------------------
1. КАКИЕ ДАННЫЕ СОБИРАЮТСЯ

Приложение обрабатывает: ИИН сотрудника, номер телефона, ФИО, должность,
отдел и филиал, историю отметок прихода и ухода, дату и время
сканирования, геолокацию в момент отметки, изображение с камеры при
сканировании QR-кода, а также биометрические данные — фотографию лица,
используемую для фото-верификации личности при отметке. Также
собираются технические данные (сведения об ошибках и работе
приложения).

2. ЗАЧЕМ СОБИРАЮТСЯ ДАННЫЕ

Данные собираются исключительно для внутреннего учёта рабочего времени:
фиксации прихода и ухода, контроля опозданий и проверки того, что
сотрудник является активным работником компании.

3. КАК ИСПОЛЬЗУЮТСЯ ДАННЫЕ

ИИН и номер телефона используются для идентификации учётной записи
сотрудника. Геолокация используется только в момент отметки, чтобы
подтвердить нахождение сотрудника у рабочей точки. Фотография лица
(биометрические данные) используется только для сверки личности в
момент отметки и не используется ни для каких иных целей. История
отметок используется для формирования табеля и аналитики опозданий.

4. КТО ИМЕЕТ ДОСТУП К ДАННЫМ

Доступ к данным имеют уполномоченные сотрудники компании-работодателя
(ТОО "AVA TECH.IO"), ответственные за учёт рабочего времени и
администрирование системы AvaTracker.

5. ПЕРЕДАЮТСЯ ЛИ ДАННЫЕ ТРЕТЬИМ ЛИЦАМ

Данные не передаются третьим лицам в рекламных или маркетинговых целях.
Приложение не содержит рекламных SDK и стороннего трекинга. Данные
обрабатываются в информационной системе работодателя.

6. ГДЕ И КАК ХРАНЯТСЯ ДАННЫЕ

Данные учёта хранятся на серверах системы AvaTracker. Передача данных
между приложением и сервером выполняется по защищённому соединению
(HTTPS/TLS). На устройстве в защищённом хранилище сохраняются только
токены сессии, ИИН, телефон и базовый профиль. SMS-коды и снимки лица
на устройстве не хранятся.

7. СРОК ХРАНЕНИЯ ДАННЫХ

Данные учёта рабочего времени хранятся в течение срока, предусмотренного
внутренними правилами компании и требованиями законодательства. Данные
сессии на устройстве удаляются при выходе из аккаунта.

8. КАК ЗАПРОСИТЬ УДАЛЕНИЕ ИЛИ ИСПРАВЛЕНИЕ ДАННЫХ

Сотрудник может самостоятельно удалить учётную запись приложения в
разделе «Профиль → О приложении → Удалить аккаунт». Удаление выполняется
немедленно и необратимо; данные учёта рабочего времени сохраняются в
основной системе компании согласно внутренним правилам и
законодательству, для повторного использования приложения потребуется
новая регистрация. Для исправления данных (ФИО, должность и т. п.)
следует обратиться к ответственному лицу по контактам ниже.

9. КОНТАКТЫ ОТВЕТСТВЕННОГО ЛИЦА

По вопросам обработки персональных данных обращайтесь к оператору
данных — ТОО "AVA TECH.IO", e-mail: info@avtch.io.
